== Description ==

Allows admins to specify verified accounts or mark users as spam accounts. Get a list of users who have been marked as spam or verified. Allows better spam account management.


== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BuddyPress Verified` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Then go to user profile >> settings >> verify page in the BuddyPress front end and choose various settings.

= Manual Installation =

1. Upload `buddypress-verified` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Then go to user profile >> settings >> verify page in the BuddyPress front end and choose various settings.
